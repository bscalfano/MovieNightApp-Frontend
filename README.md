# Movie Night App - Frontend

React application for scheduling and viewing movie nights with friends.

## Technologies
- React
- Vite
- Tailwind CSS
- Axios
- React Router
- TMDB API Integration

## Setup
1. Install dependencies: `npm install`
2. Create `.env` file with: `VITE_TMDB_API_KEY=your_api_key`
3. Update API URL in `src/services/movieNightService.js`
4. Run dev server: `npm run dev`

## Features
- Calendar and list views
- TMDB movie search with auto-fill
- Past movie history
- Search/filter movies
- Genre tags
- Custom confirmation dialogs
